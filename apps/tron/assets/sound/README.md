# Tron 3D Sound Assets

Generated local WAV effects used by `game.js`.

- `countdown-beep.wav`: countdown pulse
- `round-start.wav`: round start rise
- `turn-blip.wav`: turn input cue
- `crash.wav`: collision impact
- `round-win.wav`: win cue
- `round-draw.wav`: draw cue
- `menu-select.wav`: setup menu cue
- `engine-loop.wav`: looping light-cycle engine bed
